
## Instructions

	- To run app locally, run 'node server.js'

Bootstrap

	- Scaffolding for layouts, fonts and icons.
	- Bootstrap Datepicker for use with dates.

CSS libraries

	- Animate CSS - https://daneden.github.io/animate.css - for simple css3 animation effects


Custom JS/jQuery:

	- jQuery.selectbox
	- jQuery Date Picker

Javascript libraries used:
	- Detect Browser js - Detects which browser you are using for styling
	- Please-wait - a splash screen that runs whilst page is loading
	- Wow.js = Adds interactive animations
	- Selectbox - nicely styled selectbox
	- Smooth Scroll - for smooth scrolling of anchor links.
